# OmniReview MCP Tools v0.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox syntax for tracking.

**Goal:** Build a Python MCP server exposing 3 tools (fetch_mr_data, create_review_worktrees, cleanup_review_worktrees) that replace ~25 bash commands in OmniReview's Phases 1, 2, and 7.

**Architecture:** A single Python file (`tools/omnireview_mcp_server.py`) using FastMCP API. Subprocess calls use `asyncio.create_subprocess_exec` (argument list, safe from injection). Input validation at every tool entry. Timeouts on all subprocess calls. Registered via `.mcp.json` at repo root.

**Tech Stack:** Python 3.10+, `mcp` package (FastMCP), `asyncio`, `glab` CLI, `git`

**Spec:** `docs/specs/2026-03-26-mcp-tools-v01-design.md`
